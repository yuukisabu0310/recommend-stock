前提条件

現在の判断の前提条件は、PMI拡大・インフレ低下、政策金利高・長期金利正常、株価が200日移動平均を上回りながらボリュームが減少・ボラティリティが低い状況にあることである

想定リスク

  • 政策金利の変化が市場に与える影響、景気後退の可能性

転換ポイント

  • PMIが再加速した場合、政策金利が再び引き上げ方向に転じた場合、200日移動平均を明確に下回った状態が継続した場合

データ取得元: Yahoo Finance (yfinance), FRED API, e-Stat API

更新頻度: 毎日（GitHub Actions自動実行）

指標計算方法:

  • 移動平均: 単純移動平均（20日、75日、200日）
  • ボラティリティ: 過去30日の日次リターンの標準偏差を年率換算
  • 出来高比率: 最新出来高 / 過去30日の平均出来高
  • トレンド判定: 価格と移動平均の順序関係から判定
