前提条件

現在の判断の前提条件として、景気の拡張が続いていること、株価の長期的なトレンドが上昇していること、金融政策の影響が市場に及んでいることが含まれる。さらに、インフレ率の低下や、政策金利の水準も考慮されている。

市場環境サマリー

現在の市場環境では、PMIが50を超えており、景気の拡張が続いていると考えられる。また、株価が長期的なトレンドライン上にあることから、上昇の可能性がある。ただし、インフレ率が低下していることや、金融政策の影響により、市場の方向性が変化する可能性もある。

想定リスク

  • 景気の減速や金融政策の変更による市場の変動、インフレ率の低下が持続し、景気に悪影響を及ぼす可能性

転換ポイント

  • PMIが3ヶ月連続で50未満になった場合、株価が200日移動平均を3日連続で下回った場合、政策金利が2%以上変化した場合

データ取得元: Yahoo Finance (yfinance), FRED API, e-Stat API

更新頻度: 毎日（GitHub Actions自動実行）

指標計算方法:

  • 移動平均: 単純移動平均（20日、75日、200日）
  • ボラティリティ: 過去30日の日次リターンの標準偏差を年率換算
  • 出来高比率: 最新出来高 / 過去30日の平均出来高
  • トレンド判定: 価格と移動平均の順序関係から判定
