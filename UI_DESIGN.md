# 株式市場分析ページ UI/UX設計書（初心者向け）

## 1. ページ全体の構成

### セクション構成（上から順）

```
┌─────────────────────────────────────┐
│ ヘッダー（シンプル）                 │
```

### デザインのポイント

- **カードサイズ**: 画面幅の90%（スマホ）、最大800px（PC）
- **余白**: 上下左右に十分な余白（パディング: 32px）
- **色**: 
  - 強気: 薄い緑背景（#F0FDF4）+ 濃い緑テキスト（#166534）
  - 中立: グレー背景（#F9FAFB）+ グレーテキスト（#374151）
  - 弱気: 薄い赤背景（#FEF2F2）+ 濃い赤テキスト（#991B1B）
- **アイコン**: 
  - 方向性: 🟢（強気）/ 🟡（中立）/ 🔴（弱気）
  - リスク: ⚠️（中）/ ✅（低）/ 🚨（高）
- **フォント**: 
  - 方向性ラベル: 24px、太字
  - 判断理由: 16px、通常
  - 更新日時: 12px、グレー

## 3. 各クリック展開セクションの見出しと内容例

### セクション1: 「詳しく見る」をクリックしたとき

#### 見出し: 「なぜそう判断したか」

**内容例（初心者向けの文章）:**

```
現在の市場環境を判断するために、以下の3つの観点から見ています：

1. 株価の動き
   過去数週間、株価は全体的に上昇傾向にあります。
   これは、多くの投資家が前向きに考えていることを示しています。

2. 経済の状態
   景気を示す指標が、前向きな状態を保っています。
   ただし、一部の指標には注意が必要な部分もあります。

3. 市場の雰囲気
   取引量が増えていることから、市場への関心が高まっていることがわかります。

※ これらの情報は参考としてお使いください。
   投資判断は、必ずご自身で行ってください。
```

**デザイン:**
- 背景: 白
- 各項目: カード形式、左に番号アイコン
- 文字サイズ: 14px、読みやすい行間

---

### セクション2: 「想定されるリスク」展開

#### 見出し: 「注意しておきたい点」

**内容例:**

```
現在の判断が変わる可能性がある条件を、わかりやすくまとめました：

⚠️ 主なリスク

・景気指標が悪化した場合
  現在は前向きな状態ですが、経済の指標が悪化すると、
  市場の雰囲気が変わる可能性があります。

・大きな出来事が起きた場合
  予想外の出来事（例：政策の変更、国際情勢の変化など）が
  起きた場合、市場が大きく動く可能性があります。

・株価が急激に下がった場合
  もし株価が急激に下がり始めたら、市場の雰囲気が
  変わり始めている可能性があります。

これらの条件が見られたら、判断を見直すタイミングかもしれません。
```

**デザイン:**
- リスク項目: 黄色の背景（#FEF3C7）+ アイコン
- 各項目: カード形式、余白を多めに

### セクション5: 「分析ロジック（中級者向け）」展開

#### 見出し: 「分析の仕組み（詳しく知りたい方向け）」

**内容例:**

```
この分析は、以下のような方法で行っています：

🔍 分析の流れ

1. データの取得
   株価や経済指標のデータを、自動で取得します。

2. 指標の計算
   移動平均やボラティリティなどの指標を計算します。
   ※ 移動平均: 過去の株価の平均値
   ※ ボラティリティ: 株価の変動の大きさ

3. 総合的な判断
   複数の指標を組み合わせて、市場の状態を判断します。
   単一の指標だけで判断することはありません。

4. 文章の生成
   判断結果を、わかりやすい文章に変換します。

💡 注意点
この分析は、あくまで参考情報です。
投資判断は、必ずご自身で行ってください。
```

**デザイン:**
- 背景: 白
- 専門用語: ツールチップで説明（ホバーで表示）
- フローチャート風のアイコン配置

## 4. 初心者が「安心できる」トーンの文章例

### メインメッセージ例

**良い例（安心感がある）:**
```
「景気指標と株価の流れから、短期的にはやや前向きな状態です。
ただし、経済の状況は常に変化するため、定期的に確認することを
おすすめします。投資判断は、必ずご自身で行ってください。」
```

**悪い例（不安を煽る）:**
```
「市場は上昇トレンドにありますが、急落のリスクもあります。
早めの判断が必要です。」
```

### リスク説明の例

**良い例:**
```
「現在の判断が変わる可能性がある条件を、わかりやすくまとめました。
これらの条件が見られたら、判断を見直すタイミングかもしれません。
ただし、必ずしもこの通りになるとは限りません。」
```

**悪い例:**
```
「以下のリスクが発生した場合、市場は急落します。
早急な対応が必要です。」
```

### 免責事項の例

**良い例（短縮版、ファーストビュー）:**
```
「この情報は参考としてお使いください。投資判断は自己責任で行ってください。」
[詳細を見る ▼]
```

**展開版:**
```
この分析は、投資助言や売買指示を目的としたものではありません。

・投資判断は、必ずご自身で行ってください
・過去の実績は、将来を保証するものではありません
・この情報は、あくまで参考としてお使いください

投資にはリスクが伴います。ご自身の判断で、慎重に検討してください。
```

## 5. なぜこの構成が初心者向けなのか

### 理由1: 情報の段階的提示
- **問題**: 初心者は一度に多くの情報を見ると混乱する
- **解決**: 最初は最小限の情報だけ表示し、必要に応じて詳細を展開
- **効果**: 心理的な負担が減り、理解しやすくなる

### 理由2: 専門用語の排除
- **問題**: PMI、CPI、移動平均などの専門用語は初心者には理解困難
- **解決**: 日常的な言葉に置き換え（例：「景気指標」「株価の平均」）
- **効果**: 専門知識がなくても理解できる

### 理由3: 視覚的な理解の促進
- **問題**: 数値だけでは判断が難しい
- **解決**: アイコンと色で直感的に理解できる（🟢=強気、🔴=弱気）
- **効果**: 一目で市場の状態がわかる

### 理由4: 安心感の提供
- **問題**: 金融情報は不安を感じやすい
- **解決**: 断定を避け、「可能性があります」「参考として」という表現を使用
- **効果**: プレッシャーを感じず、冷静に判断できる

### 理由5: スマホ優先設計
- **問題**: 初心者はスマホで閲覧することが多い
- **解決**: スマホ画面を最優先に設計し、PCは後から対応
- **効果**: どこでも手軽に確認できる

### 理由6: 判断材料の提示（判断の押し付けではない）
- **問題**: 「買い」「売り」という指示は初心者には負担が大きい
- **解決**: 「判断材料」として提示し、最終判断はユーザーに委ねる
- **効果**: 自分で考える力を育て、自信につながる

## 6. 追加のUI/UX改善提案

### マイクロインタラクション

1. **カードホバー時**
   - 軽く浮き上がる（シャドウが濃くなる）
   - カーソルがポインターに変わる

2. **展開アニメーション**
   - スムーズに開閉（0.3秒のトランジション）
   - フェードインで表示

3. **ローディング状態**
   - データ取得中は、スケルトンローディングを表示
   - 「データを取得しています...」というメッセージ

### アクセシビリティ

1. **色だけで情報を伝えない**
   - アイコンとテキストで補完
   - 色覚異常の方にも理解できる

2. **フォーカス状態**
   - キーボード操作でも使いやすい
   - フォーカスリングを明確に表示

3. **読み上げ対応**
   - 適切な見出しタグを使用
   - 画像には代替テキストを設定

## 7. 実装時の注意点

### パフォーマンス
- 最初の読み込みを3秒以内に
- 画像は最適化（WebP形式など）
- 不要なJavaScriptは遅延読み込み

### セキュリティ
- データ取得元を明示
- 個人情報は一切扱わない
- HTTPS必須

### 保守性
- コンポーネント化して再利用可能に
- スタイルは一箇所で管理
- 文章は外部ファイル化（多言語対応の準備）

---

この設計により、投資初心者でも「市場の今の状態」を直感的に理解でき、必要に応じて詳細情報を確認できる、使いやすいUIを実現できます。
